## Description
A Machine Learning project that predicts whether a breast tumor is Benign or Malignant using diagnostic measurements from the Breast Cancer Wisconsin Dataset. The model is trained using a Random Forest Classifier and deployed through an interactive Streamlit web application for easy predictions.

Features

* Data preprocessing and cleaning

* Exploratory Data Analysis (EDA)

* Feature visualization using charts and correlation heatmaps

* Breast cancer classification using Random Forest

* Model serialization with Pickle

* Interactive prediction interface using Streamlit

* High-accuracy tumor classification
